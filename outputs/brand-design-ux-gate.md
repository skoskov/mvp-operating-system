# Brand / Design System / UX Flow gate — 2026-06-22

## Result

Brand / Design System / UX Flow was added as a mandatory mvp-os gate after Launch Market Research and before Memory Preflight.

## Scope

This block captures implementation-ready product and interface instructions:

- product name;
- logo direction or reference;
- slogan;
- short description;
- positioning;
- tone of voice;
- design references;
- design system;
- UX flows in text diagram format;
- core scenarios;
- edge cases;
- acceptance criteria.

## Example included

```text
Product name:
MaxCRM

Slogan:
Максимум пользы из вашей записной книжки!

Short description:
Личная CRM, которая пишет в вашем стиле. Поздравления и напоминания в нужный момент.
```
