# Optional Limited Release / Launch Plan

This is an optional gate.

Use it when the MVP is ready for a controlled test with a limited group of users or when the owner explicitly asks for GTM planning.

Do not require this plan before implementation by default.

## Purpose

Define a limited release test before a broader GTM plan.

The goal is to validate product readiness with a small audience, not to scale acquisition.

## When to use

Use this plan when:

- the MVP will be shown to a closed group;
- the owner wants a limited beta/test;
- the product needs first real-user feedback;
- launch risk is high and a small test is safer;
- there is an explicit decision to prepare GTM.

Do not use it when:

- the product is still pre-MVP;
- the first test is purely internal;
- the user did not ask for launch/GTM planning.

## Rules

- Limited release is not full GTM.
- Keep audience small and controlled.
- Do not create a large marketing plan unless explicitly requested.
- Do not scale acquisition before product, analytics, support, and known issues are ready.
- If moving from limited release to GTM requires budget, partners, paid traffic, or public promises, create a Decision Card.
