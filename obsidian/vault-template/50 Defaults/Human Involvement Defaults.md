# Human Involvement Defaults

## Do not ask the human to

- run terminal commands;
- paste logs;
- review code;
- approve long specs;
- validate file checklists;
- perform routine QA;
- choose routine libraries.

## Ask the human only for

- MVP scope;
- important technical trade-offs;
- budget/token/time approval;
- product/business logic;
- unit economics;
- risk appetite;
- examples of good/bad outputs where taste matters.
